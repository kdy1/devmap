# next.js 사용 목적

참고: 필자(`@kdy1`)는 next.js 팀원입니다.

next.js는 SSR 하려고 쓰는 게 아닙니다.
next.js는 리액트에 편의 기능을 잔뜩 끼얹은 라이브러리로 보는 게 가장 정확합니다.
편의 기능 중에 SSR이 있는 것이지, SSR을 사용할 떄만 next.js를 쓰는 게 아닙니다.

[SWC 플레이그라운드](https://play.swc.rs)나 이 사이트, 제 영문 블로그와 제가 만드는 자잘한 사이트들은 전부 next.js를 사용합니다.
그 중에 SSR을 쓰는 프로젝트는 없는데도 말이죠.
